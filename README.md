# 🌿 Bloom & Petal Haven

A front-end web development project built using **HTML5**, **CSS**, and **JavaScript**.

---

## 📌 Project Overview

Bloom & Petal Haven is a webpage designed for a neighborhood flower shop that offers beautiful flower arrangements and indoor plants. This project was developed as part of my personal initiative to apply and assess the foundational web development skills I acquired during my first year at the **Technological Institute of the Philippines (TIP) — Manila**.

Throughout my studies, I gained a thorough understanding of HTML structure, syntax, attributes, and values, along with the role that CSS and JavaScript play in enhancing the visual presentation and interactivity of web pages. This project serves as a practical demonstration of those competencies, reflecting my ability to design and implement a structured, styled, and functional webpage from the ground up.

---

## 🛠️ Technologies Used

| Technology | Purpose |
|------------|---------|
| HTML5 | Page structure and content |
| CSS3 | Styling, layout, and animations |
| JavaScript | Interactivity and cart functionality |
| Google Fonts | Typography (Cormorant Garamond, Jost) |

---

## ✨ Features

- **Product Showcase** — Displays two featured indoor plants (Peace Lily and Zanzibar Gem) with descriptions and prices
- **Add to Cart** — Interactive cart button that updates the item count in the navigation bar
- **Toast Notification** — A pop-up confirmation message appears each time a product is added to the cart
- **Hover Animations** — Smooth card lift and image zoom effects on mouse hover
- **Sticky Navigation Bar** — Navigation stays visible at the top while scrolling, with a frosted glass effect
- **Responsive Design** — Layout adjusts cleanly for both desktop and mobile screen sizes
- **Page Load Animations** — Staggered fade-in animations on the hero section and product cards
